### 컨텐츠 기반 영화 추천 플랫폼

-   영화 제목을 검색하면 비슷한 영화 10개를 추천해주는 서비스
-   검색한 영화가 데이터베이스에 없으면 검색어와 비슷한 제목의 영화를 제시하는 기능
-   컨텐츠 기반 추천 알고리즘 중 코사인 유사도 함수를 사용했습니다.
-   개발 기간 : 2021.12.13 ~ 2021.12.24
